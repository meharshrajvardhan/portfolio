Getting Started

Clone the Repository

```bash
git clone https://github.com/meharshrajvardhan/harshdevstack.git
```

Navigate to the Project

```bash
cd harshdevstack
```

Install Dependencies

```bash
npm install
```

Start Development Server

```bash
npm run dev
```

Build for Production

```bash
npm run build
```
